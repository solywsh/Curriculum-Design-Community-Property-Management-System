# Curriculum-Design-Community-Property-Management-System

## c++课程设计-小区物业管理系统

此代码为c++课程设计题，采用控制台实现。

系统可设置收费标准，物业费、电梯费和暖气费收取均根据房屋面积计算得到。主要功能包括：

- 用户管理；

- 房屋信息和业主信息的录入、修改、删除和文件导入；

- 季度费用信息的手工录入，修改、删除和文件导入；

- 按楼号+房号查询业主交费信息；

- 小区房屋维修记录的录入、删除和查询；

- 按楼号、季度统计未交费的房号，并输出房号，业主姓名，电话和费用，保存到文件中；

- 小区公告管理。
